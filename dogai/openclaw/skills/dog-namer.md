# Skill: dog-namer

สร้างชื่อภาษาไทยน่ารักสำหรับหมาจากลักษณะที่วิเคราะห์ได้

## Input
- สีขน, พันธุ์, บุคลิก, ขนาด, เพศ (จาก vision analysis)

## Output
- ชื่อ 3-5 ชื่อ พร้อมที่มาของชื่อแต่ละตัว

## Guidelines

- ชื่อควรสั้น 1-2 พยางค์ ออกเสียงง่าย
- อ้างอิงจากลักษณะของหมา เช่น สีขน, นิสัย, รูปร่าง
- ตัวอย่างชื่อดีๆ:
  - สีขนส้ม → ส้มจี๊ด, มะม่วง, ทองดี
  - ขนขาว → มิลค์, หิมะ, ขาวปุย
  - ดำ → ออบซิเดียน, มะขาม, ถ่าน
  - ตัวเล็กน่ารัก → จิ๋ว, เม็ดงา, ถั่ว
  - ขี้เล่น → แบงค์, ซน, วิ่ง
  - เชื่อง → เฟิร์น, ขนุน, ใจดี

## Format

```
ชื่อที่แนะนำ:
1. **[ชื่อ]** — [เหตุผล 1 ประโยค]
2. **[ชื่อ]** — [เหตุผล 1 ประโยค]
3. **[ชื่อ]** — [เหตุผล 1 ประโยค]
```
