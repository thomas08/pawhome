# Skill: update-narrator

เขียน narrative update อบอุ่นสำหรับ DogUpdate ส่งให้ followers

## Input
- ข้อมูลหมา (ชื่อ, พันธุ์, บุคลิก)
- รูปภาพใหม่ (ถ้ามี)
- Session/time context (เช้า/เย็น)

## Output
- Narrative 3-5 ประโยค ภาษาไทย อบอุ่น เหมือนเขียนจดหมายถึงแฟนๆ
- mood string (เช่น "สดใส", "ง่วงนอน", "ขี้เล่น", "หิว")

## Guidelines

- เล่าเรื่องราวประจำวันของหมาตัวนั้น
- สร้างความผูกพันระหว่าง follower กับหมา
- morning update → บรรยากาศเช้า, กิจกรรมเริ่มต้นวัน
- evening update → กิจกรรมในวัน, บรรยากาศผ่อนคลาย
- ไม่ซ้ำๆ กัน ทุกวัน creative ต่างกัน

## Format

```json
{
  "narrative": "วันนี้น้องส้มจี๊ดตื่นมาตั้งแต่เช้าตรู่...",
  "mood": "สดใส"
}
```

## Example

```
น้องส้มจี๊ดตื่นนอนมาพร้อมพลังงานเต็มเปี่ยมเลยค่ะ เช้านี้วิ่งเล่นในสนามหลังบ้านแล้วก็มานั่งรอขนมอยู่หน้าประตูเหมือนเดิม 😄 ดูเหมือนจะหิวมากเป็นพิเศษวันนี้ค่ะ ใครอยากมาเล่นด้วยได้เลยนะคะ น้องยินดีต้อนรับทุกคนค่ะ 🐾
```
