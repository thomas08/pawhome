# Skill: photo-describe

สร้าง caption ภาษาไทยสำหรับรูปหมา ใช้สำหรับ DogPhoto.caption

## Input
- รูปภาพหมา 1 รูป
- ชื่อหมา (ถ้ามี)

## Output
- Caption 1-2 ประโยค ภาษาไทย อบอุ่น น่ารัก

## Guidelines

- เล่าสิ่งที่เห็นในรูปอย่างอบอุ่น
- ใช้ชื่อหมาถ้ามี
- อาจมี emoji น้อยๆ
- ตัวอย่าง:
  - "น้องส้มจี๊ดนอนแดดยามเช้าอย่างมีความสุข ขนสีส้มทองระยับสวยงามมากค่ะ 🌟"
  - "ท่าทางซุกซนของน้องถั่วกำลังวิ่งเล่นในสนาม พลังงานเยอะมากเลยค่ะ 🐾"
